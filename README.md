## Run API
Walk to the folder
```
$ cd ./Backend
```

## Build container
```
$ docker-compose up -d --build
```

## Run Front
Walk to the folder
```
$ cd ./Frontend
```

## Build container
```
$ docker-compose up -d --build
```


### Build (front)
```
$ yarn run build
```

### Build (api)
```
$ go build -o ./{application}
```


### Run without docker (front)
```
$ yarn dev
```

### Run without docker (api)
```
$ go run main.go
```
